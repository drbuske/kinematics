Definition 4.1.14. Gradient Vector Fields.

The gradient of a scalar function \(f\) is a function that assigns a vector to each point, and is therefore a vector field. It is called the gradient vector field of \(f\text{.}\) In two dimensions, we denote the gradient vector field of \(f\) at \((a,b)\) by
\begin{equation*} \nabla f(a,b) = \frac{\partial f}{\partial x}(a,b)\vec{i} + \frac{\partial f}{\partial y}(a,b)\vec{j}. \end{equation*}
