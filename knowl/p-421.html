We now estimate the rate at which \(z\) changes with respect to a change in variable \(x\) while keeping \(y\) fixed at a value of 2. Using data in Table 3.1.7 gives
\begin{equation*} \frac{\partial z}{\partial x}(1,2) \approx \frac{\Delta z}{\Delta x} = \frac{3.79-4.19}{1.1-0.9} = -2 \end{equation*}
which matches the computed value from Example 3.1.2.
