Solution 1.4.3.2.1.

A. With \(x=0\) and \(y=1\text{,}\) we can find \(r\) from \(r=\sqrt{x^2+y^2}=1\text{.}\) We can find \(\theta\) from \(\tan \theta = y/x = \hbox{undefined}\text{.}\) Thus, \(\theta = \pi/2\text{.}\) Since \((0,1)\) is in the first quadrant this is correct. The polar coordinates are \((1,\pi/2)\text{.}\) B. With \(x=-1\) and \(y=0\text{,}\) we can find \(r\) from \(r=\sqrt{x^2+y^2}=1\text{.}\) We can find \(\theta\) from \(\tan \theta = y/x = 0\text{.}\) Thus, \(\theta = 0\text{.}\) Since \((-1,0)\) is in the second quadrant the correct angle must be \(180^\circ\) (\(=\pi\)) from this. The polar coordinates are \((1,\pi)\text{.}\)
