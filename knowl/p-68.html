The graph of a function of one variable is a curve in 2-space. It has a one-dimensional level set (points on a line) as seen in Figure 1.3.11. The graph of a function of two variables is a surface in 3-space. It has a two-dimensional level set (curves in a plane) as seen in Figure 1.3.12. The graph of a function of three variables would be a solid in 4-space. It has a three-dimensional level set (surfaces in space). An example of a single level set (surface) for the function \(p(x,y,z)\) of three variables that gives the atmospheric pressure at location \((x,y,z)\) is shown in Figure 1.3.13.
