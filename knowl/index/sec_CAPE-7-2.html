The Trapezoid Rule is an average of the Left and Right Riemann sums: \(T_n = \frac{L_n + R_n}{2}\) If a function is always increasing or always decreasing on the interval \([a,b]\text{,}\) one of \(L_n\) and \(R_n\) will over-estimate the true value of \(\int_a^b f(x) \ dx\text{,}\) while the other will under-estimate the integral. Thus, the errors that result from \(L_n\) and \(R_n\) will have opposite signs; so averaging \(L_n\) and \(R_n\) eliminates a considerable amount of the error present in the respective approximations. In a similar way, it makes sense to think about averaging \(M_n\) and \(T_n\) in order to generate a still more accurate approximation. Since \(M_n\) is typically about twice as accurate as \(T_n\text{,}\) we call the weighted average
\begin{equation} S_{2n} = \frac{2M_n + T_n}{3}\tag{1.3} \end{equation}
Simpson’s Rule. Note that we use ``\(S_{2n}\)" rather than ``\(S_n\)" since the \(n\) points the Midpoint Rule uses are different from the \(n\) points the Trapezoid Rule uses, and thus Simpson’s Rule is using \(2n\) points at which to evaluate the function.
