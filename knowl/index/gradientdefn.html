Definition 5.2.1. The Gradient Vector.

The gradient vector of a differentiable function \(f(x,y)\) at a point \((a,b)\text{,}\) denoted \(\nabla f (a,b)\text{,}\) is defined by
\begin{equation*} \nabla f (a,b) = \frac{\partial f}{\partial x}(a,b) \vec{i} + \frac{\partial f}{\partial y}(a,b) \vec{j}. \end{equation*}
