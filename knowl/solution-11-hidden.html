(a) First let two adjoining sides of the rectangle be the vectors \(\vec a\) and \(\vec b\text{:}\) using the vectors from \((0,0,0)\) to \((-2,0,0)\) and from \((0,0,0)\) to \((0,-1,0)\text{,}\) we have
\begin{equation*} \vec a = -2\vec{\mathit{i}}, \qquad \vec b = -\vec{\mathit{j}}. \end{equation*}
Because the area of a rectangle is given by the cross product of vectors along its sides, we can calculate an area vector as
\begin{equation*} \vec a\times\vec b = 2\vec{\mathit{k}}. \end{equation*}
However, because this is an upward pointing vector, we want the negative of this: \((-2k)\text{.}\) We could have obtained this directly by using the right-hand rule to note that \(\vec a\times\vec b\) will have an upward orientation while \(\vec b\times\vec a = -\vec a\times\vec b\) will point downward. (b) The area vector for the circle must have magnitude \(\pi r^2 = 9\,\pi\text{,}\) and must point in the positive \(y\)-direction, so \(\vec{A_2} = 9\,\pi \,\vec j\text{.}\)
