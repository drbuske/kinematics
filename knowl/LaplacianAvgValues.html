Example 4.6.14. The Laplacian and Average Values Near a Point.

The function
\begin{equation*} u(x,y)=1+e^{-x}\sin(y) \end{equation*}
has \(\frac{\partial u}{\partial x} = -e^{-x}\sin(y)\) and \(\frac{\partial u}{\partial y} = e^{-x}\cos(y)\text{.}\) So, \(\frac{\partial^2 u}{\partial x^2} = e^{-x}\sin(y)\) and \(\frac{\partial^2 u}{\partial y^2} = -e^{-x}\sin(y)\text{.}\) It follows that the Laplacian is zero:
\begin{align*} \Delta u = \mathstrut \amp \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2}\\ = \mathstrut \amp e^{-x}\sin(y) -e^{-x}\sin(y)\\ = \mathstrut \amp 0. \end{align*}
We interpret this to mean that near any point \((x_0,y_0,u(x_0,y_0))\) on the graph of \(u(x,y)\text{,}\) the average value of \(u(x,y)\) is equal to the value \(u(x_0,y_0)\text{.}\)
The vector field \(\nabla u(x,y)\) for \(u(x,y)=1+e^{-x}\sin(y)\) is divergence-free (sometimes also called solenoidal). There are no sources or sinks (see FigureÂ 4.6.16). The interested reader should try plotting this function over various domains. They will find the graph of \(u\) has no maxima or minima and the gradient vector field \(\nabla u\) has no sources or sinks.
