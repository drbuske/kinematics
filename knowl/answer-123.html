Answer 3.3.3.1.

  1. \(\frac{\partial u}{\partial t} = -ce^{x-ct}\) and \(\frac{\partial u}{\partial x} = e^{x-ct}\) so that \(\frac{\partial u}{\partial t}+c\frac{\partial u}{\partial x} = -ce^{x-ct} + c e^{x-ct} = 0\text{.}\)
  2. \(\frac{\partial u}{\partial t} = -c \cos(x-ct)\) and \(\frac{\partial u}{\partial x} =\cos(x-ct)\) so that \(\frac{\partial u}{\partial t}+c\frac{\partial u}{\partial x} = -c\cos(x-ct) + c \cos(x-ct) = 0\text{.}\)
