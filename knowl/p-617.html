A few remarks regarding notation we will use for divergence. If we write the 'del' operator as
\begin{gather*} \nabla = \frac{\partial}{\partial x} \vec{i} + \frac{\partial}{\partial y} \vec{j} + \frac{\partial}{\partial z} \vec{k} \end{gather*}
and
\begin{gather*} \vec{F} = F_1\vec{i} + F_2\vec{j} + F_3\vec{k} \end{gather*}
then we have
\begin{align*} \nabla \cdot \vec{F} = \mathstrut \amp \left( \frac{\partial}{\partial x} \vec{i} + \frac{\partial}{\partial y} \vec{j} + \frac{\partial}{\partial z} \vec{k}\right) \cdot \left( F_1\vec{i} + F_2\vec{j} + F_3\vec{k}\right)\\ = \mathstrut \amp \frac{\partial F_1}{\partial x} + \frac{\partial F_2}{\partial y} + \frac{\partial F_3}{\partial z}\\ = \mathstrut \amp div \ \vec{F}. \end{align*}
