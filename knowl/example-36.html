Example 3.1.12.

If \(f(x,y) = 5xy+ y^2\text{,}\) then \(\frac{\partial f}{\partial x}(x,y) = 5y\) and \(\frac{\partial f}{\partial y}(x,y) = 5x + 2y\text{.}\) If we wrote \(z = 5xy+y^2\text{,}\) on the other hand, then we would write \(\frac{\partial z}{\partial x} = 5y\) and \(\frac{\partial z}{\partial y} = 5x+2y\text{.}\) Such partial derivative functions are useful in that they allow us to compute many slopes at the same time. For example, since \(\frac{\partial f}{\partial x}(x,y) = 5y\text{,}\) we have \(\frac{\partial f}{\partial x}(0,1) = 5\text{,}\) \(\frac{\partial f}{\partial x}(2,1) = 5\text{,}\) and \(\frac{\partial f}{\partial x}(2,-2) = -10\text{.}\)
