The vector field \(\vec{F}\) illustrated in Figure 4.5.4 has algebraic description of \(\vec{F}(x,y,z) = -x\vec{i}-y\vec{j}-z\vec{k}\text{.}\) From this formula, we compute the divergence:
\begin{align*} div \ \vec{F} = \mathstrut \amp \frac{\partial }{\partial x}(-x) + \frac{\partial}{\partial y}(-y) + \frac{\partial }{\partial z}(-z)\\ = \mathstrut \amp -1 -1 -1 \\ = \mathstrut \amp -3. \end{align*}
at every point \((x,y,z)\) and hence also at \((0,0,0)\text{.}\) Note that we also found negative divergence in Example 4.5.3, but this computation quantifies this value. When a point located in a vector field \(\vec{F}\) such as \((0,0,0)\) has negative divergence, we call it a sink. Conversely, when a point \(P\) has positive divergence (i.e. \(div \ \vec{F}(P) > 0\)) in a vector field \(\vec{F}\text{,}\) we call it a source.
