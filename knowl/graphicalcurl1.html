Example 4.3.4. Connecting Curl Definitions.

Consider the vector field \(\vec{F}=x\vec{j}\text{.}\) A cross section of \(\vec{F}\) that lies in the plane \(z=0\) is given in Figure 4.3.5. We investigate \({\rm{curl}} \ \vec{F}(-1,1,0)\text{.}\)
If we place a small square \(C\) centered at \((-1,1,0)\) as shown in Figure 4.3.5 in the vector field \(\vec{F}=x\vec{j}\text{,}\) we can imagine it as a small paddle wheel (see Figure 4.2.14) that will turn based on the flow. In this case, the paddle wheel will turn counter-clockwise. In fact, it is when the handle of the paddle wheel points in the direction of \(+\vec{k}\) that the it spins fastest in the counter-clockwise direction (i.e. when the circulation is greatest). So geometrically, we see that \({\rm{curl}}\ \vec{F}(-1,1,0) = \alpha\vec{k}\) for some number \(\alpha>0\text{.}\)
Using Definition 4.3.3, we can compute \({\rm{curl}}\ \vec{F}(-1,1,0)\) directly. For \(\vec{F}=x\vec{j}\text{,}\) we note that
\begin{align*} {\rm{curl}} \ \vec{F} = \mathstrut \amp \nabla \times \vec{F} \\ = \mathstrut \amp \det \left( \begin{array}{ccc} \vec{i} \amp \vec{j} \amp \vec{k} \\ \frac{\partial}{\partial x} \amp \frac{\partial}{\partial y} \amp \frac{\partial}{\partial z} \\ 0 \amp x \amp 0 \end{array} \right) \\ = \mathstrut \amp 1\vec{k} \end{align*}
which aligns with what we expected using the geometric definition of curl. Note too that, rather remarkably, \({\rm{curl}}\ \vec{F}\) is the same vector at every point in this field!
