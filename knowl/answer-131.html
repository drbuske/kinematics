Answer 4.3.6.1.

  1. Based on the figure, a small paddle wheel placed at \((0,1,0)\) would spin counter-clockwise the fastest if the handle is placed in the \(\vec{k}\) direction. So \({\rm{curl}} \ \vec{F}(0,1,0)\) points in the \(\vec{k}\) direction. Similarly, a small paddle wheel placed at \((0,-1,0)\) would spin counter-clockwise the fastest if the handle is placed in the \(-\vec{k}\) direction. So \({\rm{curl}} \ \vec{F}(0,-1,0)\) points in the \(-\vec{k}\) direction.
  2. For \(\vec{F}=(4-y^2)\vec{i}\text{,}\) we note that
    \begin{align*} {\rm{curl}} \ \vec{F} = \mathstrut \amp \nabla \times \vec{F} \\ = \mathstrut \amp \det \left( \begin{array}{ccc} \vec{i} \amp \vec{j} \amp \vec{k} \\ \frac{\partial}{\partial x} \amp \frac{\partial}{\partial y} \amp \frac{\partial}{\partial z} \\ 4-y^2 \amp 0 \amp 0 \end{array} \right) \\ = \mathstrut \amp 2y\vec{k} \end{align*}
    so that \({\rm{curl}} \ \vec{F}(0,1,0) = 2\vec{k}\) and \({\rm{curl}} \ \vec{F}(0,-1,0) = -2\vec{k}\text{.}\) This aligns with what was found in (a).
