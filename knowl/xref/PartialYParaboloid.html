to determine the value of \(\frac{\partial f}{\partial y}(1,2)\) for the function \(f(x,y)=9-x^2-y^2\text{.}\)
Answer.
In the applet, enter the function \(f(x,y)=9-x^2-y^2\) and then choose \(x=1\) and \(y=2\) by using the sliders. To discover the value of \(\frac{\partial f}{\partial y}(1,2)\text{,}\) check the Illustrate \(\frac{\partial f}{\partial y}\) box.
