Checkpoint 5.1.8. Estimating the Value of \(\frac{\partial f}{\partial x}\).

Use a central difference to estimate \(\frac{\partial f}{\partial x}(1,2)\) if \(f(x,y)=9-x^2-y^2\) using Table 5.1.7.
Answer.
We now estimate the rate at which \(z\) changes with respect to a change in variable \(x\) while keeping \(y\) fixed at a value of 2. Using data in Table 5.1.7 gives
\begin{equation*} \frac{\partial z}{\partial x}(1,2) \approx \frac{\Delta z}{\Delta x} = \frac{3.79-4.19}{1.1-0.9} = -2 \end{equation*}
which matches the computed value from Example 5.1.2.
