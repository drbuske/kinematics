Example 1.1.19. Estimating CAPE using Simpson’s Rule.

In Example 1.1.8 we estimated the value of CAPE using the Midpoint Rule and data from a sounding. We found \(M_4 \approx 2007 \ \frac{J}{kg}\text{.}\) If we expand this data set (as in Table 1.1.20) from the sounding, we can then also estimate \(T_4\text{.}\) Here,
\begin{align*} T_4 = \mathstrut \amp \left[ \frac{1}{2} f(5) + f(6) + f(7) + f(8) + \frac{1}{2}f(9)\right] \cdot 1000 \ {\rm{m}} \\ = \mathstrut \amp 9.8 \ \frac{m}{s^2} \cdot \left( \frac{1}{2}\cdot \frac{5}{265.5} + \frac{5}{253} + \frac{ 12.5}{243} + \frac{10}{238} + \frac{1}{2} \cdot \frac{5}{233} \right) \cdot 1000 \ {\rm{m}} \\ \approx \mathstrut \amp 1307 \ \frac{J}{kg} \end{align*}
Simpson’s Rule then gives
\begin{equation*} S_8 = \frac{2M_4 + T_4}{3} = \frac{2\cdot 2007 + 1307}{3} \approx 1774 \ \frac{J}{kg}. \end{equation*}
