Checkpoint 6.3.12. Estimate \(\eta\).

The horizontal wind vector field near a given location is illustrated in FigureÂ 6.3.13. The square is \(100 \ m\) by \(100 \ m\) and each vector has magnitude \(10 \ \frac{m}{s} \text{.}\) Numerically estimate the vertical component \(\eta = \nabla \times \vec{v}\) of vorticity. Recall that \(\displaystyle \eta = \frac{\partial v}{\partial x} - \frac{\partial u}{\partial y}\text{.}\)
Answer.
We estimate \(\frac{\Delta v}{\Delta x} \approx \frac{0-10 \ \frac{m}{s}}{100 \ m} = -0.1 \ \frac{1}{s}\) and \(\frac{\Delta u}{\Delta y} \approx \frac{10\cos(45)-10\cos(60) \ \frac{m}{s}}{100 \ m} \approx -0.02 \ \frac{1}{s}\text{.}\) It follows that the vertical component of vorticity is \(\eta = \frac{\partial v}{\partial x} - \frac{\partial u}{\partial y} \approx -0.1 - 0.02 = -0.12 \ \frac{1}{s}\text{.}\)
