Example 2.2.5. Fitting a Sinusoid to Geopotential Heights.

At \(45^{\circ}N\) latitude, 500 mb geopotential heights \(Z\) as a function of degrees longitude \(t\) (see FigureÂ 2.2.6) are periodic with period \(T=360\text{.}\) Our objective is to model \(Z\) using a trigonometric polynomial (2.8) with \(m=1\) (i.e. a sinusoid). In this situation the fundamental frequency is \(\omega = \frac{2\pi}{360}\text{.}\)
When \(m=1\text{,}\) we look for the best periodic function
\begin{equation*} f(t) = a_0 + a_1\cos(\omega t) + b_1\sin(\omega t) \end{equation*}
to fit the data \(\left\{ (t_n, y_n) \right\}\text{.}\) We compute:
\begin{align*} a_0 = \mathstrut \amp \frac{\sum y_n}{N} \approx 5656.45 \\ a_1 = \mathstrut \amp \displaystyle \frac{2}{N}\sum y_n \cos( \omega t_n) \approx 44.73 \\ b_1 = \mathstrut \amp \displaystyle \frac{2}{N} \sum y_n \sin( \omega t_n) \approx 12.79 \end{align*}
Thus, since \(\omega = \frac{2\pi}{360}\text{,}\) we model the geopotential heights \(Z\) by the sinusoid
\begin{equation*} f(t) = 5656.45 + 44.73\cos\left( \frac{2\pi}{360} t\right) + 12.79\sin\left(\frac{2\pi}{360} t\right). \end{equation*}
Plotting this function against our data illustrates the best fit we can make to this data without using additional harmonics \(2\omega\text{,}\) \(3\omega\text{,}\) \(4\omega\text{,}\) \(\ldots\text{.}\)
