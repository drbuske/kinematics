Checkpoint 6.2.12. Circulation from Wind II.

Determine the circulation \(\Gamma\) about a \(300 \ {\rm{km}} \times 200 \ {\rm{km}}\) rectangle where the (wind) velocity vector \(\vec{v}\) is easterly and decreasing linearly at a rate of \(2.5 \ \frac{m}{s}\) per \(100 \ {\rm{km}}\) toward the north. That is, \(\vec{v} = -(10-y/40 \frac{m}{s})\vec{i}\text{.}\)
Answer.
\begin{align*} \displaystyle \Gamma = \mathstrut \amp \int_C \vec{v} \cdot d\vec{r} \\ = \mathstrut \amp (-10 \ \frac{m}{s} \ \vec{i})(300 \ {\rm{km}} \ \vec{i}) + (-10-y/40 \ \frac{m}{s} \ \vec{i})(200 \ {\rm{km}} \ \vec{j}) \\ \mathstrut \amp + (-5 \ \frac{m}{s} \ \vec{i})(-300 \ {\rm{km}} \ \vec{i}) + (-10-y/40 \ \frac{m}{s} \ \vec{i})(-200 \ {\rm{km}} \ \vec{j})\\ = \mathstrut \amp -3,000,000 \ \frac{m^2}{s} + 0 + 1,500,000 \ \frac{m^2}{s} + 0\\ = \mathstrut \amp -1,500,000 \ \frac{m^2}{s}. \end{align*}
